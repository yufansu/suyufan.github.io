How to Build Homepage

Posted on 2017-08-15

Basic Env Requirements

  • Github Account
  • Git
  • Node.js
  • Hexo

Install Git

  • Windows: Download and Install git
  • Linux (Ubuntu): $ sudo apt-get install git

使用Windows的用户可以通过git中自带的git bash执行以下适用与linux系统的命令。

Install Node.js

  • Windows: Download and Install Node.js
  • Linux (Ubuntu):
    • $ wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh
    • Restart the terminal
    • $ nvm install stable

Install Hexo

  • $ npm install -g hexo-cli

Get Your Github Account and Repo

  1. Sign up for Github
  2. Create a new reposity
  3. Set the repo name to {username}.github.io

Build Your Own Homepage with Hexo

Set up the folder

  • $ hexo init <folder-name>
  • $ cd <folder-name>
  • $ npm install

Build on localhost

  • hexo generate
  • hexo server
  • view it on http://localhost:4000/
  • Now you can see the website generated from templates in Hexo.

Connect to Github Repo

  • create a folder called .deploy
  • clone the repo to it, $ git clone https://github.com/{username}/{username}.github.io.git

  • create a shell script to deploy the website

  • change mode of deploy.sh, $ chmod +x deploy.sh

  • Now view the result on your website, {username}.github.io
